# Rigol Oscilloscope Controller

โปรแกรมสำหรับควบคุมและดึงข้อมูลจากออสซิลโลสโคป (Oscilloscope) ของ Rigol ผ่านโปรโตคอล VISA โดยมี Graphical User Interface (GUI) ที่พัฒนาด้วย Tkinter ช่วยให้ผู้ใช้สามารถเชื่อมต่อ ส่งคำสั่ง อ่านค่าการวัด และดึงกราฟสัญญาณ (Waveform) ออกมาแสดงผลและบันทึกเป็นรูปภาพได้ง่ายๆ

## คุณสมบัติหลัก (Features)
- **การเชื่อมต่ออุปกรณ์:** ค้นหาพอร์ต VISA ที่เชื่อมต่ออยู่ทั้งหมด และเลือกเชื่อมต่อ/ยกเลิกการเชื่อมต่อกับเครื่องออสซิลโลสโคปได้
- **การส่งคำสั่ง SCPI:** มีช่องสำหรับพิมพ์คำสั่ง SCPI ไปยังเครื่องโดยตรง และแสดงผลตอบกลับในช่อง Console
- **การอ่านค่าการวัด (Measurement):** สามารถเลือกช่องสัญญาณ (Channel 1-4) และประเภทการวัดที่ต้องการ เช่น VPP, VMIN, VMAX, PERiod, FREQuency เพื่อดึงค่ามาแสดงผล
- **การแสดงผลกราฟสัญญาณ (Waveform):** สามารถดึงข้อมูลกราฟสัญญาณ (Waveform) จากช่องสัญญาณที่กำหนด (Channel 1-4) มาพล็อตกราฟบนหน้าจอผ่านไลบรารี Matplotlib 
- **บันทึกรูปภาพ:** สามารถบันทึกกราฟสัญญาณที่แสดงผลอยู่เป็นไฟล์รูปภาพ (PNG) ได้

## ความต้องการของระบบ (Requirements)
โปรดติดตั้งไลบรารีต่อไปนี้ก่อนใช้งานโปรแกรม:
- `pyvisa`: สำหรับการสื่อสารผ่านโปรโตคอล VISA
- `pyvisa-py`: VISA backend (ใช้งานในโค้ด `@py`)
- `matplotlib`: สำหรับพล็อตกราฟสัญญาณ Waveform
- `tkinter`: สำหรับสร้าง GUI (ปกติมาพร้อมกับ Python อยู่แล้ว)

คำสั่งสำหรับติดตั้งไลบรารี (หากยังไม่มี):
```bash
pip install pyvisa pyvisa-py matplotlib
```

## วิธีการใช้งาน (How to Use)
1. รันไฟล์ `Rigol_Oscillascope_Controller.py` เพื่อเปิดโปรแกรม
2. **การเชื่อมต่อ:** 
   - กดปุ่ม **Refresh** เพื่อค้นหาอุปกรณ์ที่ต่ออยู่
   - เลือกอุปกรณ์จาก Dropdown menu (เช่น `USB0::...`)
   - กดปุ่ม **Connect** เพื่อเชื่อมต่อ เมื่อสำเร็จปุ่มจะเปลี่ยนเป็นสีเขียว
3. **การส่งคำสั่ง SCPI:**
   - พิมพ์คำสั่งในช่อง SCPI Command แล้วกด **Send** 
   - หากเป็นคำสั่ง Query (มี `?`) โปรแกรมจะแสดงผลลัพธ์ใน Console ขวามือ
4. **การอ่านค่า (Measure):**
   - เลือก Channel และประเภทการวัด (เช่น VPP) แล้วกด **Read Value**
5. **การดึงกราฟ (Waveform):**
   - เลือก Channel ที่ต้องการดึงกราฟ แล้วกด **Read Waveform** (กราฟจะใช้เวลาสักครู่ในการดึงข้อมูลและพล็อต)
   - เมื่อกราฟแสดงแล้ว สามารถกด **Save Image (PNG)** เพื่อบันทึกรูปกราฟได้

## โครงสร้างโค้ด
- `OscillascopeController`: คลาสจัดการการเชื่อมต่อ, การส่ง/รับคำสั่ง (SCPI), และการดึงข้อมูล Waveform สื่อสารกับเครื่องด้วย `pyvisa`
- `GraphicUserInterface`: คลาสจัดการหน้าต่าง GUI ด้วย `tkinter` ประกอบด้วย Frame การตั้งค่า, การเชื่อมต่อ, หน้าจอพล็อตกราฟด้วย `matplotlib.backends.backend_tkagg` และส่วนของ Console Output

## หมายเหตุ
- โปรแกรมนี้ใช้ VISA backend แบบ `@py` ทำให้ไม่จำเป็นต้องติดตั้ง NI-VISA เพื่อใช้งานพื้นฐาน แต่หากมีปัญหากับการมองเห็นอุปกรณ์ USB อาจพิจารณาติดตั้ง NI-VISA หรือใช้อุปกรณ์ backend อื่นๆ ของ PyVISA เพิ่มเติม
- การดึงข้อมูล Waveform จะดึงความละเอียดที่ 1200 จุด (Points) ในโหมด NORMal
